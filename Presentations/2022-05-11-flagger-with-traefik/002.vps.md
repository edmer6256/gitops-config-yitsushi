## Create a Kind cluster


* Expose 80 and 443 ports (for fun).
* Worker nodes are good.

```yaml
---
kind: Cluster
name: flagger
apiVersion: kind.x-k8s.io/v1alpha4
nodes:
- role: control-plane
extraPortMappings:
- containerPort: 30000
  hostPort: 80
  listenAddress: "0.0.0.0"
  protocol: tcp
- containerPort: 30001
  hostPort: 443
  listenAddress: "0.0.0.0"
  protocol: tcp
- role: worker
```
